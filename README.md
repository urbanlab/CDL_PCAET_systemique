# Projet PCAET – Modélisation Systémique ÉcoRénov’

## Description
Ce projet vise à analyser la dynamique du dispositif ÉcoRénov’ de la Métropole de Lyon dans le cadre du PCAET, à l’aide d’une approche de dynamique des systèmes.

Un modèle Stock–Flux a été développé sous Stella afin de simuler différents scénarios de politiques publiques et d’identifier les principaux leviers permettant d’accélérer la rénovation énergétique des logements.

## Objectif
Évaluer l’impact de différents leviers d’action (financiers, techniques, sociaux) sur :
- le flux de rénovation des logements
- la consommation énergétique
- les émissions de GES à l’horizon 2030.

## Outil utilisé
- Stella Architect (modélisation et simulation)

## Livrables
- Rapport d'Analyse Systemique
- Guide utilisateur du modèle
- Présentation finale
- Tableau des leviers d’action prioritaires
